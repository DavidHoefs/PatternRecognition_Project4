ZIP file contains: 
Project4.py - The python program which trained, tested, and exported the trained model.
test.py - The file which will be used to test on YOUR dataset. This file takes the Name of the directory path containing test images and reads in the trained model, outputs classification results
pcaFit.pkl - the saved PCA model which is used in test.py
TrainedModel.pkl - The trained SVM model which is used in test.py

TO RUN:
Run test.py and enter in the path to the directory containing your dataset. Classification will be printed to the console, and to a text file names printout.txt
